### Preserving Abkhaz (аҧсуа бызшәа) — in the open

I build open, reproducible resources for **Abkhaz**, a Northwest Caucasian language that
UNESCO lists as *vulnerable* and that has very few modern, openly-licensed learning materials.

🌿 **Main project → [abkhaz-language-open](https://github.com/1977-code/abkhaz-language-open)**
An open course + language-data toolkit: a reproducible corpus→frequency pipeline (CC0),
a CEFR **A0–A2** methodology, draft textbook chapters, a language-standard guide, and a
bilingual site. *Code MIT · content CC BY 4.0 · data CC0.*

**Why it matters:** the Northwest Caucasian family has already lost **Ubykh** (its last
fluent speaker died in 1992). Open, reusable materials are a small hedge against that for
Abkhaz.

**How to help:** native speakers, linguists, developers, and translators are all welcome —
even a single corrected word is valuable. See the project's `CONTRIBUTING` guide.
